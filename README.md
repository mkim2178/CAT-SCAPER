# CAT-SCAPER

## Requirements
- Python 3.10 - 3.12
- Pygame library

## Setup Instructions

1. Create a virtual environment (recommended):
```bash
python -m venv venv
```

2. Activate the virtual environment:
    - Windows:
    ```bash
    venv\Script\activate
    ```

- macOS/Linux:
    ```bash
    source venv/bin/activate
    ```

3. Install Pygame:
```bash
pip install pygame
```

or just 
```bash
pip install -r requirements.txt
```

4. Run the project:
```bash
python main.py
```
